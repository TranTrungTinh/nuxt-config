## Build Setup

**Requires Node.js 8+**

``` bash
# install dependencies
npm install # or yarn

# serve in dev mode, with hot reload at localhost:3000
npm run dev

# build for production
npm run build

# serve in production mode
npm start

# validate code with ESLint (with Prettier)
npm run lint

# validate andf fix with ESLint (with Prettier)
npm run lintfix

```
